# Brainfuck

[https://en.wikipedia.org/wiki/Brainfuck](https://en.wikipedia.org/wiki/Brainfuck)

## install

sudo npm i -g coffeescript

cd /usr/local/bin  

ln -s /opt/node/current/lib/node_modules/coffeescript/bin/coffee

## run

./bin/bf examples/hello-world.bf

## build the bf.js

npm run build
